# LuaDatapack

Allows lua scripts to be ran in Minecraft.

## Usage

Create a new datapack with the `lua` subfolder. For example: `data/foo/lua/bar.lua`

## Commands

### `/lua`

Usage: `/lua <name>`

Example, to run `data/foo/lua/bar.lua`: `/lua foo:bar.lua`
